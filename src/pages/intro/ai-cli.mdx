##### Core API

- <InlineCode>awaitWorld</InlineCode> awaits a [World component](https://github.com/rob-myers/the-last-redoubt/blob/master/src/projects/world/World.jsx#:~:text=export%20default-,function%20World,-(props) "@new-tab") with specified key.
- <InlineCode>click</InlineCode> converts clicks and touches into world coordinates.
- <InlineCode>look</InlineCode> makes an NPC look.
- <InlineCode>nav</InlineCode> computes a navpath for an NPC.
- <InlineCode>npc</InlineCode> configure NPCs and World.
- <InlineCode>spawn</InlineCode> spawns an NPC.
- <InlineCode>track</InlineCode> follows an NPC using panning and zooming.
- <InlineCode>view</InlineCode> pan and/or zoom.
- <InlineCode>walk</InlineCode> makes an NPC walk.

##### Loops

<InlineCode>goLoop andros</InlineCode> makes Andros walk where you click.

```bash
goLoop () {
  click |
    filter 'x => x.meta.nav && (!x.meta.ui || x.meta.go)' |
    nav $1 |
    walk $1
}
```

<InlineCode>lookLoop andros</InlineCode> makes Andros look where you click.

```bash
lookLoop () {
  click |
    filter 'x => !x.meta.nav && !x.meta.do' |
    look $1
}
```
